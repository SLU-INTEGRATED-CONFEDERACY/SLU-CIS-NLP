# Tuwali–English Lexicon

## Project Description
**A comprehensive digital dictionary preserving the Tuwali language(Kiangan) of the Ifugao region**

This project presents a trilingual lexicon documenting the Tuwali language, an indigenous language spoken in the Ifugao province of the Philippines, specifically from Kiangan. The lexicon serves as a valuable resource for language preservation, linguistic research, and cultural documentation, providing detailed entries with morphological analysis, English and Tagalog translations, and cultural context.

## c. Lexicon Structure
The CSV file contains the following columns:

| Column Name | Description | Example |
|-------------|-------------|---------|
| **Tuwali Words** | The Tuwali word or phrase (with affixes if present) | `a-ahhu` |
| **Root Word** | The base/root form of the Tuwali word | `ahu` |
| **prefix** | Prefix attached to the root (if any) | `a-` |
| **infix** | Infix inserted into the root (if any) | `-in-` |
| **suffix** | Suffix attached to the root (if any) | `-na` |
| **English Translation** | English equivalent or translation | `puppies/dogs` |
| **Tagalog Translation** | Tagalog equivalent or translation | `mga tuta/aso` |
| **Meaning** | Detailed explanation, usage notes, or cultural context | `many dogs` |
| **Part of Speech** | Grammatical category of the word | `Noun` |

## d. How to Use This Lexicon

### 1. For Language Learners
- Look up Tuwali words and find English/Tagalog equivalents
- Study word formation through root words and affixes
- Use the **Meaning** column for contextual understanding and cultural notes
- Filter by **Part of Speech** to study specific grammatical categories

### 2. For Researchers
- Analyze morphological patterns (prefixes, infixes, suffixes)
- Compare semantic fields across three languages
- Use for linguistic or anthropological studies of Ifugao language and culture
- Study language contact phenomena between Tuwali, English, and Tagalog

### 3. For Developers
- Load the CSV into Python, R, or JavaScript for analysis
- Build a simple translation tool or flashcard app
- Convert to JSON for web or mobile app integration
- Create APIs for language processing applications
