# Bolinao Lexicon Project

This repository contains a **lexicon** developed as part of an academic project.  
The lexicon documents words, meanings, and example usages for linguistic analysis, preservation, and reference.

## Lexicon Fields (CSV Columns)

Each row in the lexicon corresponds to one lexical entry.  
The columns in the CSV file are defined as follows:

| Column Name | Description |
|------------|-------------|
| **word** | The main Bolinao word (lemma) |
| **part_of_speech** | Grammatical category of the word |
| **meaning_english** | English definition of the word |
| **meaning_filipino** | Filipino definition of the word |
| **sample_bolinao** | Example sentence in Bolinao |
| **sample_english** | English translation of the example sentence |
| **upos** | English Universal parts of speech tag |

## How to Use the Lexicon

### 1. Academic Research
- Use the lexicon as a reference for linguistic and language documentation studies.
- Analyze vocabulary, grammatical categories, and semantic meanings.

### 2. Language Learning
- Study the meanings and example sentences to understand proper word usage.
- Compare Bolinao, English, and Filipino translations.

### 3. Data Analysis
- Open the CSV file using spreadsheet software or programming tools.
- Sort or filter entries by part of speech or word.

### 4. Preservation and Documentation
- The lexicon serves as a digital record for language preservation and future reference.
