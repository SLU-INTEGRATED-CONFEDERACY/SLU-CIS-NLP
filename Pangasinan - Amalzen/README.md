# Pangasinan Lexicon (NLP Pangasinan - Lexion)

## Overview
This repository contains a CSV file with a lexicon of Pangasinan words, their meanings, translations, morphological information, and other linguistic details. The lexicon is intended for use in natural language processing (NLP) research, language documentation, and educational purposes.

## File Description
- **NLP Pangasinan - Lexion - pangasinan.csv**: The main lexicon file. It includes the following columns:
  - `word`: The Pangasinan word or entry.
  - `meaning`: Explanation or definition of the word.
  - `translation`: English or Tagalog translation.
  - `source`: Source of the entry (e.g., dictionary reference).
  - `morphology`: Morphological analysis in JSON format (root, processes, etc.).
  - `root`: The root form of the word.
  - `processes`: Morphological processes applied (e.g., reduplication, affixation).
  - `type`: Type of morphological process (if any).
  - `meaning` (process): Meaning of the morphological process.
  - `partial chunk`, `form`, `normalized_form`: Additional morphological details.

## Usage
The CSV can be opened with spreadsheet software (Excel, LibreOffice Calc, Google Sheets) or processed programmatically (Python, R, etc.). It is suitable for linguistic analysis, lexicon building, and NLP applications.
